codebook
========

The features selected for this database come from the accelerometer and gyroscope 3-axial raw signals tAcc-XYZ and tGyro-XYZ. Then only measurements on the mean and standard deviation are extracted. The variable names have the form V$x$.mean, where $x$ represents the feature number, which can be seen in feature.txt.

The activity variable can have 6 values, LAYING, SITTING, STANDING, WALKING, WALKING_DOWNSTAIRS and WALKING_UPSTAIRS, and the value of the subject variable is inside the interval [1,30], and represents the ID number of the person who performed the activity. The number of rows in the data set are 180, one row for each activity and subject.
